# Physical Mail Automation Platform

An automated web-based solution designed to seamlessly bridge digital workflows with physical mail delivery. The platform enables users to create and manage direct mail campaigns through an intuitive interface, offering built-in letter templates, the ability to upload custom designs, or edit content directly using Canva integration. 

A comprehensive dashboard provides real-time visibility into campaign history, current progress, and delivery status. Users can schedule mailings in advance, monitor campaign performance, and track recipient engagement through QR code analytics, allowing them to identify whether recipients have scanned and interacted with the mailed content. 

This end-to-end solution streamlines the creation, distribution, and measurement of physical mail campaigns from a single centralized platform.

## 🚀 Live Demo
The frontend is deployed and accessible at: [Physical Mail Automation](https://physical-mail-automation.netlify.app/)

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6+)
  - Hosted on **Netlify** for fast, edge-optimized performance.
- **Backend:** FastAPI (Python 3.10+)
  - Robust, asynchronous REST API handling payload validation, background workers, and mail integration routing.

## ✨ Features

- **Campaign Management:** Create, organize, and manage direct mail campaigns from a centralized platform.
- **Customizable Letter Creation:** AChoose from professionally designed templates, upload custom artwork, or edit designs seamlessly through Canva integration.
- **Comprehensive Dashboard:** Access a user-friendly dashboard to view campaign history and monitor active mailings.
- **Bulk Mailing Automation:** Efficiently manage and distribute large mailing lists while maintaining accuracy and scalability.
